Overview: 
----------------------------------------------------------------------------------
This application is a client/server structured application using the SodaCloud framework. The client is an Android app and the server is a Java server built upon Jetty. The application implements an observer pattern for report object. The client can create and upload report to server. The client can also register listener on the reports so that when there is updates to the reports, the client can get notified automatically.

SodaCloudMaintenanceClient is client.
SodaCloudMaintenanceServer is server.
SodaCloudMaintenanceTest is test.
